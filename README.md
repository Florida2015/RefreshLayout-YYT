
根据设计的动画效果可以将整个动画分为6步

1. 画一个初始的默认矩形
2. 矩形向两侧扩展为两个梯形
3. 两个梯形由中间向外侧缩减
4. 绘制"M"的两个"腿"和两个"臂"(由无到有的动画效果)
5. 缩减"M"的两个"腿"到两个正方形小点，缩减两个"臂"到一个正方形小点
6. 初始化Loading的三个正方形小点的位置，然后一次更改其Y轴位置达到加载效果
